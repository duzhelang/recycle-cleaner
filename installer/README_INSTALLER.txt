To generate the setup installer with uninstall support:

1) Install Inno Setup 6 (https://jrsoftware.org/isinfo.php)
2) Open installer\RecycleCleaner.iss in Inno Setup
3) Click Build -> Compile
4) Output setup exe will be created inside the installer output folder.

This setup supports:
- Program Files installation
- Start menu group
- Optional desktop shortcut
- Windows uninstall entry
